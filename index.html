常见问题

短链接是否有访问次数限制？

我们的短链接不限制访问次数。免费版仅对单个短链的有效期（30 分钟）做了限制；企业版及 Pro 版用户可根据业务需求自定义有效期，实现长期稳定可用。

是否支持私有化部署和企业专属域名？

我们支持私有化部署，可运行在企业自有服务器上，并使用企业自有域名。系统可与短信服务商对接，支持灵活定制业务需求。 需要注意的是：在电信运营商的短信发送中，如短链的备案主体与企业主体不一致，可能会导致短信发送失败或被拦截。私有部署方案可有效避免此类问题，确保短信顺利下发。

是否支持自定义域名？

自定义域名功能仅在企业版及以上版本中提供，允许您使用自己的品牌域名生成短链接。

可以查看访问者的设备信息吗？

企业 Pro 版支持详细的访问分析，包括用户设备、操作系统、浏览器和地理位置等。

支持 API 接入吗？

企业版和 Pro 版均开放 RESTful API，支持自动化批量生成短链接和数据同步。
